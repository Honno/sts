Reference for ``sts-pylib`` 
============================

The repository for ``sts-pylib`` is available on `GitHub
<https://github.com/Honno/sts-pylib>`_,
and a demonstration of running these randomness tests
over the ``getrandbits()`` method from Python's ``random`` module
is available on `Kaggle
<https://www.kaggle.com/justhonno/overly-simplistic-randomness-testing-demo>`_.
